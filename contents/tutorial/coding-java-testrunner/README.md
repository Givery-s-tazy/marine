## **[App.java](src/main/java/track/App.java) (編集可)**
### `add(int x, int y)`
`x` と `y` を足し算した値を返します。
[Main.java](src/main/java/track/Main.java) (編集不可)から呼び出されます。

#### パラメータ
- `int x`: 足し算する整数
- `int y`: 足し算する整数

#### 返り値
- 足し算した値

## 入力のフォーマット
* \\(x\\):
  * 足し算する整数。
  * \\(0\\)～\\(99\\) の整数。
* \\(y\\):
  * 足し算する整数。
  * \\(0\\)～\\(99\\) の整数。

## 実行例
### 例1
#### add の引数の値
* \\(x\\): 2
* \\(y\\): 3

#### 標準出力の結果

``` console
5
```

2 + 3 は 5 です。
